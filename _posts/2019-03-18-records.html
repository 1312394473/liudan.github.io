一：调试工具

1. iOS之Safari调试webView/H5页面
2. 手机端debugger查看

第一步：安装spy-debugger： windows：npm install spy-debugger -g mac：sudo npm install spy-debugger -g

第二步：手机和PC保持在同一网络下（比如同时连到一个Wi-Fi下）

第三步：命令行输入spy-debugger，按命令行提示用浏览器打开相应地址。

第四步：设置手机的HTTP代理，代理IP地址设置为PC的IP地址，端口为spy-debugger的启动端口(默认端口：9888)。

参考资料：debugger操作介绍

二：端口占用操作

lsof -I:9001 根据指定的端口查看对应的进程情况 获取pid

Kill -9 pid 找到需要杀掉的进程，执行
