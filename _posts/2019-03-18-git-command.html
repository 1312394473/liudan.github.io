常用基础git操作命令

git命令 git说明
git status 查看状态
git fetch 获取总分支
git branch (-r) 查看所有远程分支
git clear 清空
ls 查看文件
rm -Rf * 删除所有文件
git clone url 获取对应的git地址源码
git checkout 对应分支名称 切换到指定的分支
git checkout . 撤回所有修改
git stash 修改的代码进行缓存
git pull 远程代码更新
git stash pop 恢复缓存代码
git commit -am 'desc' 进行commit操作
git push(git push origin --all) 进行push操作
git log 查看提交的commitid（取前6位） 查看日志
git reset --hard commitid（取前6位） 重置当前代码到commitid版本
git merge --no-ff 指定版本 合并某个版本(要把什么代码合并到什么分支上，先要切换到该分支上在 进行以上的操作)
git checkout -b 分支名称 创建并切换到指定的分支
git branch -d 分支名称 删除某个分支
rz 不会覆盖已存在的文件
rz -y 强制覆盖所有文件
unzip 文件名称 解压文件
